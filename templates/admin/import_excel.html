Import Documents from Excel
Excel File Format

Your Excel file should contain the following columns:

  • Document ID (required) - Unique identifier (e.g., DOC-01, SAF-001)
  • Document Name (required) - Full title of the document
  • Format (optional) - File format (.docx, .xlsx, .pdf, etc.) - default: .docx
  • Status (optional) - Document status (e.g., Active, Draft, Tracked — file not held, Active — Pending Signature)
  • Distribution (optional) - Distribution level (e.g., Internal, External (AASB), External (BDA))
  • Owner (optional) - Document owner/party (e.g., Internal, Calmic, BDA)
  • Description (optional) - Brief description or summary of the document
